# Musical PowerShell - Make your scripts sound awesome!

## Abstract

"I heard you're in a band. What instrument do you play?"
"PowerShell!"

PowerShell can be a tool for many things - But have you ever considered using it to create music? The new MIDI 2.0 standard and the latest and greatest Microsoft MIDI tools opens up an entirely new world for our favourite shell! Join me in the loudest session in all of PSConf, looking through some of the cool, weird, and most likely unexpected things we can do with MIDI in- and output and a bit of PowerShell magic.

After this session you will see how much PowerShell rocks. Literally.


## Session info

The reaper projects use the following:
- Reaper DAW - https://www.reaper.fm/ - for keyboard sounds
- SWS / S&M extension - https://sws-extension.org/ - for project management and opening .RPL file
- Ugritone drums - https://ugritone.com/
- Surge synth - https://surge-synthesizer.github.io/
- Deputy strings - https://www.fullbucket.de/music/deputy.html
- 99 sounds Upright piano - https://99sounds.org/free-piano-vst/
