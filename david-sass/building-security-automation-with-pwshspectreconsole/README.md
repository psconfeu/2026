# Building security automation with PwshSpectreConsole

## Abstract

Linux has all these cool-looking Terminal UI applications but none which I could use because I don't have a use case for them. But what if I could create my own TUI apps, so I could be as cool as those Linux people who rock in the terminal?

In this session, we'll combine the power of PwshSpectreConsole with the Microsoft Graph API in order to not only automate incident management in Microsoft Defender XDR but also to showcase how awesome PowerShell can be when using the already available building blocks from the community.

Why attend? 
Because dealing with cloud APIs in a TUI requires many more considerations than building an offline application, and in this session, we'll talk about background API calls and caching required to enable responsiveness.
