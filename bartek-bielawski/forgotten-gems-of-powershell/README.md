# Forgotten Gems of PowerShell

![Forgotten Gems/ Hania Bielawska](Gem.png)

You can find "slides"/ demos in separate [GitHub repo](https://github.com/bielawb/powershell-gems/tree/master).

## Abstract

Whether you are a seasoned PowerShell user or just starting, getting comfortable and efficient in PowerShell can be the difference between being good and excellent at what you do. In this session, I’m going to remind you of some fantastic features that the language offers that have been there virtually forever, that you either never heard of before or forgot.
