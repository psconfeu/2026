*Cybersecurity Consultant @TDC Erhverv*

Morten is a automation specialist who loves to automate the hard stuff using code. Whether it's Infrastructure-as-Code, pipeline automation or cloud native platforms, with PowerShell, Terraform and Python as his tools of choice he works through the clouds of Azure on a daily basis.

He works at TDC Erhverv in Denmark as a Cybersecurity Consultant, and blogs about Azure, PowerShell, Terraform and various IT adventures on https://mynster9361.github.io/. He enjoys open source, spending time with the family wife + 2 kids, plays LoL.

Maintainer of open source tools such as the PowerShell modules LeastPrivilegedMSGraph, ActionableMessages, MSGraphPermissions and andrewplaclips.
